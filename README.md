# peanuts-love-letter

Interactive love letter website for Burcu, built as a single HTML file with vanilla JavaScript.

## Features

- Animated intro message for Peanut Number Zero and Honeybee
- Memory Lane with clickable relationship moments
- Pinkie Promise interactive lock/unlock element
- Tiny in-bed Dance Party mode animation
- Countdown to Burcu's birthday (March 4)
- Hand-holding simulator (hands come together)
- Love letter section
- Photo placeholders with local image upload and persistence
- Bee and heart themed romantic animations
- LocalStorage persistence for visits and interactive state

## Run locally

Open `index.html` directly in your browser.

## Files

- `index.html`: Entire website (HTML + CSS + JS)
- `README.md`: Project documentation

## Publish to GitHub

```bash
gh repo create peanuts-love-letter --public --source=. --description 'Interactive love letter for Burcu' --push
```
